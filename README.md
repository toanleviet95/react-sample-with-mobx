### Tech
- [Babel][babeljs] - Compiling JS
- [Webpack][webpack] - Bundle and dev server
- [Hot loader][react-hot-loader] - React Hot Loader

### Command

- Install
```bash
npm install // or yarn
```

- Start
```bash
npm start // or yarn start
```

- Build
```bash
npm run build // or yarn build
```
